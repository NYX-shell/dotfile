# NYX dotfile
An personal dotfile config for Linux/macOS/Windows

# requirement
- curl (99% build-in)

# usage
Linux / macOS / WSL
```SH
curl -Lso installer.sh https://github.com/NYX-shell/dotfile/raw/main/installer.sh ; sh installer.sh ; rm installer.sh
```
Windows PowerShell (no WSL)
```SH
irm https://github.com/NYX-shell/dotfile/installer.sh | irm
```
